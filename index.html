Course Browser

Browse UMich classes in a clear and organized way, along with a dynamic Google Maps display for selected classes. Web developed using Python2.7, HTML5, CSS3, and deployed using Heroku. Web framework using Flask and Jinja2, uses Google Maps API and UMich classes API. Working condition, no longer maintained. Email me for access to the code!
